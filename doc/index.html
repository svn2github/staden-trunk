Staden Package - Introduction

The Staden Package is a set of tools covering sequence assembly, editing and analysis. New releases can be found at the SourceForge package site.


Local documentation:

Program Summary Lists the program names along with a brief description and a link to the full documentation, where available.
Package Manual A copy of the full online manual for the full package. This is also available in PDF format too.
Introductory manual This is much shortened copy of the full manual, consisting primarily of introductions to the tools available. Also available in PDF format.
Scripting manual This describes the scripting language built into Gap4. Based on Tcl, this manual describes the Gap4 extensions to the language rather than the Tcl syntax itself. Also available in PDF format.
